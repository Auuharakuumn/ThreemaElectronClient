# Threema Electron Client

## To Use

```bash
# Clone this repository
git clone https://github.com/Auuharakuumn/ThreemaElectronClient
# Go into the repository
cd ThreemaElectronClient
# Install dependencies
npm install
# Run the app
npm start
```
